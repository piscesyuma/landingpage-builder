## Installation

1. **Clone the repository**
```bash
git clone https://github.com/piscesyuma/WebsiteBuilder.git
cd website-builder
npm install
npm run dev
```
2. **Install dependencies**
```sh
npm install
# or
yarn install
# or
pnpm install
```
3. **Set up environment variables**
```sh
cp .env.example .env.local
```
4. **Run the development server**
```sh
npm run dev
# or
yarn dev
# or
pnpm dev
```
